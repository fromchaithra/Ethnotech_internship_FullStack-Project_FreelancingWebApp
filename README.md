# 🚀 FreelanceHub – Full Stack Freelancing Platform

FreelanceHub is a full-stack web application that allows users to register as **Students** or **Admins**, where:
- Students can browse and apply for jobs
- Admins can post new job opportunities

This project is developed using **Spring Boot (Backend)** and **React (Frontend)**.

---

## 🛠️ Tech Stack

### Frontend
- React (Vite)
- JavaScript
- CSS
- Axios
- React Router

### Backend
- Spring Boot
- Java
- REST APIs
- JPA (Hibernate)

### Database
- MySQL

---

## ✨ Features

- 🔐 User Authentication (Login & Register)
- 👨‍🎓 Role-Based Access (Student / Admin)
- 📄 Job Listing (Dynamic from Backend)
- ➕ Admin can Post Jobs
- 🎯 Students can Apply for Jobs
- 🔄 Real-time Frontend & Backend Integration
- 🎨 Responsive UI with modern card design
